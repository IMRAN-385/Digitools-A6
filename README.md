# DigiTools - Premium Digital Tools Marketplace

## 📌 Project Overview

**DigiTools** is a modern AI-powered digital tools marketplace that brings together premium AI tools, design assets, templates, and productivity software in one place. 

It helps creators, professionals, and businesses to supercharge their digital workflow with high-quality products like AI writing tools, design template packs, stock assets, automation toolkits, and more.

**Live Link:** [https://digitools-work.netlify.app](https://digitools-work.netlify.app)

---

## ✨ Key Features

- Curated collection of premium digital products
- Beautiful product cards with Best Seller, Popular, and New badges
- Subscription-based pricing plans (Starter, Pro, Enterprise)
- Simple 3-step "Get Started" process
- Fully responsive and modern gradient UI
- Transparent and flexible pricing
- 14-day free trial option
