# DigiTools - Supercharge Your Digital Workflow

DigiTools is a cutting-edge platform for creators, designers, and professionals to access premium digital tools, AI assets, and templates in one place. This project is a modern web application built with React and advanced styling frameworks to provide a seamless and premium user experience.

## 🚀 Technologies Used

- **React 19**: Modern React with optimized performance.
- **Tailwind CSS 4**: The latest utility-first CSS framework for ultra-fast styling.
- **DaisyUI 5**: A comprehensive component library built on top of Tailwind.
- **JavaScript (ES6+)**: Clean, modern logic and state management.
- **React-Toastify**: Professional, dynamic alerts for user feedback.
- **JSON Data**: Structured product management for easy updates.
- **Vite**: Ultra-fast build tool for local development.

## 🎯 Core Features

### 1. Dynamic Cart Management
Experience a fully interactive shopping cart system. Add products with a single click, see real-time badge updates in the Navbar, and manage your selections easily. The Cart section provides a clean, single-column overview of your items with the ability to remove selections and proceed to a simulated checkout.

### 2. Seamless Section Toggling
The main application area features an intuitive toggle system. Users can switch between the **Products Gallery** and the **Cart Overview** without page reloads, ensuring a fluid browsing experience. The "Products" view is set as default to showcase the available digital tools immediately.

### 3. Premium Interactive Design
Every part of DigiTools is designed with a premium aesthetic in mind:
- **Responsive Layout**: Pixel-perfect across desktop, tablet, and mobile devices.
- **Hover Effects**: Subbtle micro-interactions on product cards and buttons create a living interface.
- **Real-Time Feedback**: `react-toastify` provides instant notifications for all key actions—Adding to Cart, Removing Items, and Checkout.
- **Optimized UI**: High-contrast, vibrant purple themes paired with clean typography for maximum clarity.
